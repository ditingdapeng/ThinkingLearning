## 评分标准

| 分数 | 抽象层 | 具象层 | 费曼表达 |
|------|-------|-------|---------|
| **1分** | 只复述内容 | 无法举例 | 大量术语 |
| **2分** | 归纳法提炼 | 同领域举例 | 较模糊 |
| **3分** | 有模式无原理 | 能类比 | 较清晰 |
| **4分** | 有演绎链条 | 跨领域举例 | 清晰 |
| **5分** | 第一性原理清晰 | 多领域迁移 | 12岁孩子能懂 |

## 核心概念公式

```
刻意练习
  专业化 = 识别结构 + 练习结构 + 内化结构

归纳法 vs 演绎法
  归纳：案例 → 规律（概率性，隐含连续性假设）
  演绎：原理 → 结论（必然性，可靠）

第一性原理
  自确定的原起点 → 演绎推理链条 → 可靠结论

费曼学习法
  选择概念 → 简单解释 → 识别空白 → 简化重构
```
