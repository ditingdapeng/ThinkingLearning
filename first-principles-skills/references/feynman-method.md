# 费曼学习法指南

> 本文档详细说明费曼学习法的操作方式及其与苏格拉底式提问的结合。

---

## 核心理念

**费曼学习法**：如果你无法用简单的语言向一个12岁的孩子解释一件事，说明你并没有真正理解它。

**本质**：教是最好的学。通过"输出"来检验"输入"的质量。

---

## 四个步骤

### 步骤1：选择概念

明确要学习的核心概念。本系统13节课的核心概念映射：

| 课程 | 核心概念 | 要能解释的问题 |
|------|---------|---------------|
| 1 | 刻意练习 | 为什么高手和业余选手差距这么大？ |
| 2 | 思维模型 | 为什么有的人学什么都快？ |
| 3 | 读书方法 | 为什么读了很多书却没用？ |
| 4 | 第一性原理 | 什么是真正可靠的推理？ |
| 5 | 哲学基石 | 为什么需要第一性原理？ |
| 6 | 发现方法 | 如何找到第一性原理？ |
| 7 | 牛顿世界观 | 为什么工业革命能成功？ |
| 8 | 达尔文世界观 | 为什么创新难以计划？ |
| 9 | 政治经济 | 为什么计划经济失败了？ |
| 10 | 巴菲特 | 为什么价值投资有效？ |
| 11 | 孙正义 | 为什么他能预见科技趋势？ |
| 12 | 马斯克 | 为什么他能做到"不可能"？ |
| 13 | 非连续性 | 为什么成功者往往突然被淘汰？ |

### 步骤2：用简单语言解释

**AI引导追问**：
- "如果向一个12岁的小孩解释，你会怎么说？"
- "不用任何术语，用生活中的例子来解释"
- "假设对方完全没有背景知识，你怎么能让他理解？"

**评判标准**：

| ❌ 错误信号 | ✅ 正确信号 |
|------------|------------|
| 使用原文术语 | 生活化语言 |
| 需要前置知识 | 从零开始解释 |
| 抽象概念堆砌 | 具象化举例 |
| "这个比较复杂" | 简单直接 |

### 步骤3：识别理解空白

当学生无法简单解释时，就暴露了理解的盲点。

**常见空白信号**：
- "这个比较复杂，一两句话说不清" → 没有真正理解
- 绕回原文的表述方式 → 只是记忆，不是理解
- 使用模糊的指代词（"那个"、"就是"） → 概念不清晰
- 越解释越复杂 → 缺少核心抓手

**AI行为**：
1. 记录学生卡壳的具体概念
2. 追问具体哪个环节不清楚
3. 引导学生回去重读相关部分
4. 记录到 `user_think_log.md`

### 步骤4：简化和重新表达

用类比、比喻、故事来重新组织理解。

**AI引导追问**：
- "能用一个生活中的比喻来解释吗？"
- "如果画一张图，你会怎么画？"
- "用一个完全不同领域的例子来类比"
- "如果写成一句话的标题，你会怎么写？"

## 与苏格拉底式提问的结合

```
┌──────────────────────────────────────────────────────────┐
│                    学习循环                               │
│                                                          │
│   费曼表达 ─→ 暴露空白 ─→ 苏格拉底追问 ─→ 重读课程        │
│       ↑                                    │             │
│       └────────────────────────────────────┘             │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

| 环节 | 方法 | 目的 |
|------|------|------|
| 第一轮表达 | 费曼 | 检验是否理解 |
| 发现问题 | 苏格拉底 | 追问推理基石 |
| 回去重读 | 自学 | 填补理解空白 |
| 再次表达 | 费曼 | 简化重新构建 |

**关键区别**：
- **费曼**：要求简单→检验理解深度
- **苏格拉底**：追问前提→检验推理可靠性

两者结合 = 既要深度理解，又要逻辑严密。

## 费曼表达评分

| 分数 | 标准 | 示例 |
|------|------|------|
| 1分 | 只能使用原文术语，无法简化 | "第一性原理就是使用演绎法的推理基石" |
| 2分 | 有简化尝试，但仍需大量背景知识 | "就是找到那个不可再分的前提" |
| 3分 | 能用生活语言解释，偶尔用术语 | "就像盖房子要先打地基，推理也需要最底层的真理" |
| 4分 | 完全生活化表达，有一个好例子 | "想象推理是一座塔，每层楼都建在下一层上，最底层那块不会塌的地基，就是第一性原理" |
| 5分 | 12岁孩子能懂，有多个精妙类比 | "我姐姐总说'因为妈妈说的'，但如果我问'妈妈为什么这么说'，一直问下去，最后会找到一个谁都没法否认的道理，那就是第一性原理" |

## 提问模板

### 引导简单表达

- "假设我是你的12岁表弟，完全没学过这些，你怎么给我讲？"
- "不用'第一性原理'这个词，你能解释它的意思吗？"
- "用做饭/打游戏/坐地铁这种生活场景来举例"
- "如果只能说一句话，这节课的核心是什么？"

### 识别理解空白

- "你刚才说了三遍'就是'，但我还是没听懂，能再具体点吗？"
- "你用了X这个术语，X到底是什么意思？"
- "我注意到你卡住了，是哪个环节不太清楚？"
- "你能把刚才说的那段再简单讲一遍吗？"

### 引导类比

- "这个模式像生活中的什么？"
- "如果要画一张图来表示，你会画什么？"
- "历史上有没有类似的例子？"
- "能用一个完全不同领域的事来类比吗？"

## 记住

费曼学习法的本质：**教是最好的学**。

- 能讲给"小白"听懂 = 真正内化了知识
- 讲不清楚 = 还没真正理解
- 越简单的解释 = 越深刻的理解
